# Python-3.14.0-Django-5.2-E-commerce-Application
This is an E-Commerce Aplication with Administration panel for free use. DB is SQLite.

1) Before launching the project:
```bash
$python -m venv env
```
2) Then activate it:
```bash
$source env/Scripts/activate
```
2) Run server:
```bash
$python manage.py runserver
```
